🚀 Flask Task Manager with Real-Time Analytics
This is a full-stack Task Management application built to demonstrate backend API development, database integration, real-time WebSocket communication, and data analytics.

🌟 Key Features
Authentication: Secure User Login & Registration.

RESTful APIs:Complete CRUD operations for tasks (Add, Update, Delete, Get).

PostgreSQL Database: Robust relational database schema for users and tasks.

Analytics Module: Real-time calculation of Total, Pending, Completed, and Completion Percentage using Pandas and NumPy.

Real-Time Updates: Instant UI updates across all logged-in sessions without page refresh using WebSockets (Flask-SocketIO).

Modern UI: Clean, responsive dashboard built with Bootstrap 5.

🛠️ Tech Stack
Backend: Python, Flask, Flask-SQLAlchemy, Flask-SocketIO

Database: PostgreSQL

Data Analytics: Pandas, NumPy

Frontend: HTML5, CSS3, JavaScript (Fetch API), Bootstrap 5
